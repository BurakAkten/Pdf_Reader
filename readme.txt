This code reads pdfs and then extract the text in these pdfs. 
The directory named results include the txt files of the pdf files.
You should not delete the results directory.
You can also put pdf files into the directory named pdfs

To run this code :
-You have to install Python2x on your Ubuntu.
-you have yo install PyPDF2 with pip install PyPDF2

The usage of the code is :
$python2 pdf_reader.py <pdf_files_directory_path>
